### Comments

##### Single line comment
```java
  // Comment
  
  // Valid
  // // comment
  // /* comment */
```

##### Multi line comment
```java
  /*
  * Comments
  */
  
  // Valid
  /*
  * // comment
  */
  
  // Invalid
  /*
  * /* comment */
  */
```
#### Multi line comment using unicode \u002F is unicode for / and \u002A is unicode for *
```java
\u002F\u002F Single Line Comment
\u002F\u002A
    Multi Line Comments
    Multi Line Comments
\u002A\u002F
```
